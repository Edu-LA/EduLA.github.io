Siren's Sorrow

About This Project

Last Night Jammin' is an analog horror game created during a halloween game jam where the theme was "one last night".

The game follows a security guard tasked with protecting a group of participants during a game jam. The goal was to create a game with a dark atmosphere, while incorporating all five optional constraints, which were:

  • The game could only use arrow keys
  • The game had to be black and white
  • All audio had to be recorded using smartphones
  • The game couldn't include any faces
  • Backgrounds had to be based on real locations near the jam site

My Role

  • Level Design:
    Since the we had to use real locations, my job as a level designer was analysing the surroundings of the jam site, and create a map with interest points, then take out pictures of said interest points and edit them to put in engine.
  • UI Elements:
    Created UI elements to support clarity of information and player feedback.

This project presented me with a new challange. Using a real life ambient that I am familiarized with and think about it in a completely different way. Besides the skills that every game jam helps improve, such as teamwork and prototyping speed, it provided me an oppurtunity to see something I've grew accostumed to in a completely differnt light.
